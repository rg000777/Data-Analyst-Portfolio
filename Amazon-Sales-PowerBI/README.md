# 🛒 Amazon Sales Performance Dashboard

## 📋 Business Problem
Analyze 50,000+ Amazon sales records to identify 
revenue trends across regions and product categories.

## 🛠️ Tech Stack
SQL (MySQL) → Excel → Power BI Dashboard

## 📊 Key Insights
- ₹5.47M total revenue across 4 regions
- Electronics led all product categories
- Revenue decline trend identified over order timeline
- Asia & North America top performing regions
