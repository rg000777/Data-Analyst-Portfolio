# 🤖 AI & Workforce Disruption Dashboard 2020–2026

## 📋 Business Problem
Analyze 15,000 job records to measure AI's impact 
on global employment across industries from 2020 to 2026.

## 🛠️ Tech Stack
SQL (MySQL) → Excel → Power BI Dashboard

## 📊 Key Insights
- AI replacement scores spiked sharply post-2024
- Medium-risk jobs dominate across all industries
- Finance & Technology most disrupted sectors
- Avg AI Replacement Score: 46.16
